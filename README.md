# Gestión de Gastos (Tkinter + SQLite)

## Objetivo
Permitir a [público objetivo] registrar y consultar ingresos/gastos por cuentas y categorías.

## Funcionalidades básicas
- CRUD de Usuarios, Cuentas, Categorías y Transacciones
- Búsquedas, validaciones mínimas, y manejo de errores
- Interfaz en Tkinter con pestañas y tabla (Treeview)

## Estructura
- `gestiondegastos.db` (SQLite)
- `python "disenos de formularios.py"` (UI)
- `info.txt` (enlaces a GitHub y Trello)

## Cómo ejecutar
```bash
python "disenos de formularios.py"
